记录一些sql的知识

2020年01月28日 790 字 大概 3 分钟

记录点sql的一些知识

SQL数据查询方法

1、普通查询：

SELECT * FROM <表名>;

查找表内所有内容。

2、条件查询：

SELECT * FROM <表名> WHERE (条件表达式);

查找表内所有符合条件表达式的数据。

3、投影查询：

SELECT aa ab,bb bc,cc cd FROM <表名> WHERE (当然也可以加条件表达式);

查找指定列（aa,bb,cc,…）的数据，aa ab这里起了个别名，结果集的表格列名ab代替了aa。

4、分页查询：

SELECT * FROM <表名> LIMIT n OFFSET x;

假设有100行数据，n= 10, 就是分成10页，每页显示10行，x=0查询第一页，x= 10查询第二页…

可以简写成SELECT * FROM <表名> LIMIT x, n；

5、聚合查询：

SELECT (聚合函数) FROM <表名> ;

  1. SUM()
  2. AVG()
  3. MAX()
  4. MIN()
  5. COUNT()

5、多表查询：

表1：cat，表2：dog

SELECT cat.age,cat.color,dog.age,dog.color FROM cat,dog,...;

就是从多张表格查询；

举例：如果表名为neigehevi这种较长的不便于重复使用，就简写成n。

6、连接查询：

SELECT c.type c.age d.type d.age FROM cat c INNER JION dog d on cat.type = dog.type;

从cat和dog中查询type一样的，输出到结果集表格中。

SQL数据修改方法

1、插入

INSERT INTO <表名> （字段1,字段2...）VALUES (值1，值2...);

也可以插入多行。

2、更新

UPDATE <表名> SET 字段1=值1,字段2=值2...WHERE (条件表达式);

3、删除

DELETE <表名> WHERE (条件表达式);

更新和删除语句不加WHERE的时候会改动整个数据表格，要谨慎。

摘录一些好用的SQl语句

1、插入或替换

如果我们希望插入一条新记录（INSERT），但如果记录已经存在，就先删除原记录，再插入新记录。此时，可以使用REPLACE语句，这样就不必先查询，再决定是否先删除再插入：

REPLACE INTO <表名> (字段1，字段2,字段3...) VALUES (值1,值2,值3...);

若字段1=值1的记录不存在，REPLACE语句将插入新记录，否则，当前字段1=值1的记录将被删除，然后再插入新记录。

2、插入或更新

如果我们希望插入一条新记录（INSERT），但如果记录已经存在，就更新该记录，此时，可以使用INSERT INTO ... ON DUPLICATE KEY UPDATE ...语句：

INSERT INTO <表名> (字段1，字段2,字段3...) VALUES (值1,值2,值3...) ON DUPLICATE KEY UPDATE 字段2=值2,字段3=值3...;

若字段1=值1的记录不存在，INSERT语句将插入新记录，否则，当前字段1=值1的记录将被更新，更新的字段由UPDATE指定。

3、插入或忽略

如果我们希望插入一条新记录（INSERT），但如果记录已经存在，就啥事也不干直接忽略，此时，可以使用INSERT IGNORE INTO ...语句：

INSERT IGNORE INTO <表名> (字段1，字段2,字段3...) VALUES (值1,值2,值3...);

若字段1=值1的记录不存在，INSERT语句将插入新记录，否则，不执行任何操作。
