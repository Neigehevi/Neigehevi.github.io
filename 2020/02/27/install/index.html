apache/vue/angular/react配置使用遇到的问题及解决

2020年02月27日 429 字 大概 1 分钟

apache/vue/angular/react的配置使用时，有遇到一些小问题，记录下

1.apache：

我是使用xampp软件的apache功能

http://localhost/默认打开的是：

但是，要访问的是本地服务器文件：

解决：进入Config中的Apache(httpd.conf),将LoadModule dir_module modules/mod_dir.so注释掉。

在以下框架进行安装前，安装好NodeJS。

2.Vue

1.安装vue-cli 脚手架

首先要安装Vue的脚手架，在cmd中输入npm install -g vue-cli安装。

2.新建一个vue项目模版

接着，进入要存放项目的文件夹，输入vue init webpack myProject创建项目模版，myProject是项目名称，可以自定义。

3.安装依赖包

在项目目录下输入cnpm install，安装部分完成了。

使用VScode时，打开项目文件夹，在终端中输入npm run dev即可进行运行项目。

3.Angular

1.安装angular cli

在cmd中输入npm install -g @angular/cli安装。

2.新建一个angular项目模版

进入要存放项目的文件夹，输入ng new myProject创建项目模版，myProject是项目名称，可以自定义。

这里我遇到到问题了，cmd一直无法下载完成模版文件,

解决：在Github里下载angular的quickstart模版可以进行代替。

使用VScode时，打开项目文件夹，在终端中输入npm start即可进行运行项目。

4.React

1.安装react-native-cli

在cmd中输入npm install -g react-native-cli安装。

2.新建一个react项目模版

进入要存放项目的文件夹，输入react-native init myProject创建项目模版，myProject是项目名称，可以自定义。

使用VScode时，打开项目文件夹，在终端中输入react-native start即可打开开发服务器。
